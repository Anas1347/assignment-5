1. What is the Document Object br Model (DOM)? (Click for Answer)

The DOM is a tree-like structure representing a webpage, enabling scripts to manipulate its content and layout.

2. How do you select an element from the DOM? (Click for Answer)

I can select a DOM element using methods like getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector() or querySelectorAll() .

3. What is event delegation in the context of the DOM, and why is it useful? (Click for Answer)

Event delegation uses a parent element to handle events from its children, reducing event listeners and supporting dynamic content.

4. How do you manipulate an element's attributes and styles using the DOM? ? (Click for Answer)

I can manipulate an element's attributes using setAttribute() and getAttribute(), and modify styles with element.style.property.
